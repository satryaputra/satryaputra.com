## Examples

### Custom Animation Speed

Adjust the `duration` to make the travel effect faster or slower.

```tsx
<StarsTravel svgOptions={{ duration: 1.5 }} />
```

### With Foreground Content

The component accepts `children` which are rendered on top of the animation.

```tsx
<StarsTravel className="min-h-72 rounded-xl border">
  <div className="flex h-full flex-col items-center justify-center p-6 text-center">
    <h3 className="text-2xl font-bold tracking-tight">Warp Speed</h3>
    <p className="text-muted-foreground">
      Content layered perfectly above the stars.
    </p>
  </div>
</StarsTravel>
```

### As an Absolute Background

Use it as a background layer for an existing container.

```tsx
<div className="relative h-64 w-full overflow-hidden rounded-lg bg-black text-white">
  <StarsTravel className="pointer-events-none absolute inset-0" />
  <div className="relative z-10 p-8">
     {/* Your Content here */}
  </div>
</div>
```

## API Reference

| Prop                  | Type        | Default | Description                                           |
| :-------------------- | :---------- | :------ | :---------------------------------------------------- |
| `children`            | `ReactNode` | -       | Content to be rendered above the animated background. |
| `className`           | `string`    | -       | Additional CSS classes for the root container.        |
| `svgOptions`          | `object`    | `{}`    | Animation configuration for the SVG stars.            |
| `svgOptions.duration` | `number`    | `3`     | The time in seconds for one full animation cycle.     |
